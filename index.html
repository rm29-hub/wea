In this activity, I learned about export and import, object, destructuring, template literal, and arrow function. Export/import helps us share code between different JavaScript files. The export will let you bring out functions or data from your js file to your main js file, and the main js file will accept it by using import. Objects are similar to arrays but use curly brackets to hold different types of data. Destructuring is one way to access the object, other way to access it is using dot notation. Template literals let us create dynamic strings easily. Lastly, arrow function it is feature of ES6, it is shorter than the regular function and it has arrow "=>" sign.
